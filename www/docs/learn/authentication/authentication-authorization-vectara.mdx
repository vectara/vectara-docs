Vectara’s authentication and authorization framework secures access to our 
platform. Whether you are an administrator configuring corpora, a developer 
integrating APIs, or an end user querying with an application, this section 
provides the tools and strategies to control access effectively. From API keys 
for quick setups to OAuth 2.0 for enterprise-grade security, Vectara balances 
usability with robust protection.


## Who uses Vectara?

Our platform serves diverse personas: Platform Admins manage systems, 
Developers (like Dev Team Leads, App Developers, and ML Engineers) build and 
test integrations, Client Apps bridge to Application End Users, and leaders 
(Technical and Business) oversee strategy. This blueprint guides you through 
their access needs:

```
--------------------------------------------------
| Technical/Business Leaders (Oversight)         | 
--------------------------------------------------
+- - - - - - - - - - - - - - - - - - - - - -+ 
| Vectara Users                             |
|  +----------------+  +------------------+ |
|  | Platform Admin |  | Developers       | | 
|  |                |  | (Leads, App, ML) | |
|  |                |  |                  | |         
|  |  Manage        |  |  Query/Index     | |
|  +----------------+  +------------------+ |
|         |                 |               |
|         v                 v               |
|  +----------------------------+           |
|  | Vectara Platform          |            |
|  | (Identities & Corpora)    |            |
|  +----------------------------+           |
|         | API Requests                    |
|         v                                 |
|  +------------------+                     |
|  | Client App       |                     |
|  +------------------+                     |
+- - - - - - - - - - - - - - - - - - - - - -+
            | End User Auth/Access
            v
  +-----------------------+ 
  | Application End User  |
  |                       |
  +-----------------------+
```

with Vectara directly, Client Apps connect to Application End Users, and 
leaders oversee it all. Dive into the topics below for details.*
